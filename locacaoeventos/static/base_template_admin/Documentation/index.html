HTML Structure

This template has two sections, the header, the sidebar, and the main container. 

Header Contains:
  • Search Box
  • Notifications, Pending Tasks and Messages
  • User Functions (Inbox, Profile, Calender, Tasks)
Sidebar Contains:
  • Logo
  • Main Navigation
  • Container contains all the necessary codes

 

CSS Files

Bootstrap Stylesheet

Bootstrap stylesheet is the foundation stylesheet which is utilized to reset the browser inconsistencies as well as styling the bootstrap components. This stylesheet is required to be included in your page.
